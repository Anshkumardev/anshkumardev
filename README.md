🔗 [**YOLO-Based Thermal Object Detection for ADAS**](https://github.com/Anshkumardev/YOLO-Object-Detection-and-Classification-for-ADAS)

🔗 [**Android OCR & Summarization App**](https://github.com/Anshkumardev/AI-Scanner)
